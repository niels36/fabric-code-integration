# Fabric Code Integration

This repository demonstrates code integration patterns in Microsoft Fabric using different notebook types.

## Contents

- **MyNotebook.Notebook** - PySpark notebook showcasing code integration in Fabric
- **MyPythonNotebook.Notebook** - Python notebook showcasing code integration in Fabric
- **NB_Utils.Notebook** - Sample integration code

## Prerequisites

To run the notebooks in this repository, you will need:

- A Microsoft Fabric workspace
- An additional Lakehouse item
- An Environment item configured for your workspace

## Getting Started

1. Import the notebooks into your Fabric workspace
2. Configure the required Lakehouse and Environment items
3. Run the notebooks to see code integration examples
